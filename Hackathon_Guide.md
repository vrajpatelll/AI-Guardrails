# 🚀 Hackathon Participant Guide

Welcome to the hackathon! This guide covers the things you need to get started and submit your project.

## AWS Bedrock API Access

* Each team will receive one AWS Bedrock API key, which will be provisioned **only to the Team Lead** by **Nitin Chavda from the IT Team**.
* The API key comes with **$15 worth of AWS Bedrock credits**.
* Team Leads will receive (or may have already received) an email from **Nitin Chavda** containing the API key and the usage SOP.
* A dashboard link will also be shared to monitor API key usage and remaining credits.
